# SlyCreator Homebrew Tap

Official Homebrew tap for macOS apps by [SlyCreator](https://github.com/SlyCreator).

## Setup

```bash
brew tap SlyCreator/tap
```

---

## Available Apps

### SpeakClaw
System-wide voice-to-text for macOS.

```bash
brew install --cask speakclaw
```

| | |
|---|---|
| **Version** | 1.0.3 |
| **Requires** | macOS Ventura or later, Apple Silicon |
| **Homepage** | https://speakclaw.com |

---

## Update

```bash
brew upgrade --cask speakclaw
brew upgrade --cask netvow
brew upgrade --cask grindkube
```

## Uninstall

```bash
brew uninstall --cask speakclaw
brew uninstall --cask netvow
brew uninstall --cask grindkube
```
